# Tabs Project

A simple HTML/CSS/JavaScript demo that switches between tab contents when you click the tab buttons.

## Features

- Click a tab to show its matching content
- The first tab is active by default
- Clean separation of structure, style, and logic

## Project files

```text
Tabs/
├── index.html
├── style.css
└── script.js
```

- `index.html` — the main page structure and links to the CSS and JavaScript files
- `style.css` — styles for the tabs and content area
- `script.js` — tab-switching behavior

## How to run

### Option 1: Open directly in your browser

1. Go to the `Tabs` folder.
2. Double-click `index.html`.
3. The page will open in your default browser.

### Option 2: Use Live Server in VS Code

1. Open the `Tabs` folder in VS Code.
2. Install the **Live Server** extension if you do not already have it.
3. Right-click `index.html` and choose **Open with Live Server**.

## Notes

- No package installation is required.
- Make sure `index.html`, `style.css`, and `script.js` stay in the same folder so the links work correctly.
